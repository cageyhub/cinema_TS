# cinema
To run this application, execute the following commands:

Install:
```
$ npm install
```
Start the development:
```
$ npm run start
http://localhost:3001
```  
Make Build:
```
$ npm run build
```
Install JSON Server:
```
$ npm install -g json-server
```
Start JSON Server:
```
$ json-server --watch db.json
http://localhost:3000
```
